기존 Export와의 차별화
Analysis

Dashboard Export Report는 기존 export 기능들과 범위와 목적이 다릅니다:

기존: Milestone PDF / Pass Rate PDF

  • 범위: 특정 Milestone 또는 전체 통계
  • 대상: Run 결과 상세 또는 차트
  • 용도: 특정 테스트 사이클 결과 보고
  • 위치: project-runs, stats 페이지

신규: Dashboard Export Report

  • 범위: 단일 프로젝트 전체 현황
  • 대상: 프로젝트 요약 + 최근 Run + TC 현황
  • 용도: 프로젝트 상태 보고 (매니저/이해관계자용)
  • 위치: project-detail 페이지
핵심 차별점: Dashboard Export는 "프로젝트 전체 상태 요약 보고서"입니다. 개별 Run이나 Milestone이 아닌, 프로젝트 레벨에서 현재까지의 테스트 진행 상황을 한 눈에 보여주는 executive summary 성격의 문서입니다.

Export 포맷 및 동작 설계
Feature Design

A. 클릭 시 동작: 포맷 선택 드롭다운

Export Report 버튼 클릭 시 바로 다운로드하지 않고, 작은 드롭다운으로 포맷을 선택합니다:

포맷아이콘설명우선순위
PDF Report ri-file-pdf-line 시각적 보고서 (차트 포함), 매니저/이해관계자용 P0
CSV Data ri-file-excel-2-line Raw 데이터 테이블, 추가 분석용 P1

B. PDF Report 포함 내용

프로젝트의 현재 상태를 executive summary로 정리합니다:

페이지 1: 프로젝트 요약 (Executive Summary)

  • 프로젝트명, 생성일, 마지막 활동일
  • 총 Test Cases 수, 총 Runs 수, 총 Results 수
  • Overall Pass Rate (도넛 차트): Passed / Failed / Blocked / Skipped / Not Tested
  • Priority별 TC 분포 (bar chart): Critical / High / Medium / Low

페이지 2: 최근 Test Runs 현황

  • 최근 10개 Run 테이블: Run명, Milestone, 시작일, 진행률, Pass Rate
  • 각 Run의 결과 분포 미니 바
  • 활성 Run vs 완료 Run 비율

페이지 3: Test Case 현황

  • TC Lifecycle 분포: Draft / Active / Deprecated
  • Priority별 현황 요약
  • 최근 추가/수정된 TC Top 10

PDF Footer

각 페이지 하단: "Generated by Testably — {날짜} {시간}" + 페이지 번호

티어 게이팅 제안

PM 의견: Milestone PDF export가 Starter 티어 이상으로 제한되어 있으므로, Dashboard Export Report도 동일하게 Starter (tier 2) 이상으로 제한하는 것이 일관성 있습니다. Free 사용자가 클릭 시 Upgrade 모달을 표시합니다.
