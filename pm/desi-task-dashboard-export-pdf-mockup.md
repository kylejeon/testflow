# Desi 디자인 목업 지시서: Dashboard Export PDF — 8페이지 Executive Report

## 개요

CEO 확정. 현재 3페이지 기본 PDF를 8페이지 C-Level Executive Quality Report로 업그레이드합니다.
jsPDF 직접 드로잉(클라이언트 사이드)으로 구현되므로, 각 페이지의 레이아웃을 **mm 단위**로 설계합니다.

**기획안 참고:** `pm/pm-plan-dashboard-export-pdf-enhancement.html`
**세션 ID:** `local_a2550f2f-b198-4d4b-910e-2f21259cda13`

**디자인 목업 총 8건을 제작해주세요** (각 페이지별 1건씩).

## 디자인 시 참고사항

1. **jsPDF 직접 드로잉**: 이 PDF는 html2canvas가 아닌 jsPDF API(`rect`, `line`, `circle`, `text`)로 직접 그립니다. 따라서 목업은 mm 단위 좌표 기반으로 설계되어야 합니다.

2. **NotoSansKR 한글 폰트**: 한글 지원을 위해 NotoSansKR을 임베드합니다. 파일 크기 최적화를 위해 Regular + Bold만 포함합니다.

3. **동적 콘텐츠 고려**: Run 수, TC 수, 팀원 수에 따라 테이블 행 수가 변합니다. 최대 행 수를 초과하면 "... and N more" 또는 자동 페이지 추가로 처리합니다.

4. **색상 일관성**: 앱 내 Status 색상(passed=#16A34A, failed=#DC2626 등)과 PDF 색상(passed=RGB(16,163,127) 등)이 약간 다릅니다. PDF에서는 인쇄 최적화된 RGB를 사용합니다.

5. **티어 게이팅**: Free 사용자는 P1(축소)+P2+P4만 접근 가능, Professional+ 사용자만 전체 8페이지 접근 가능. 목업은 Professional+ 기준(풀 버전)으로 제작합니다.
