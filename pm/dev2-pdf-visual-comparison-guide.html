전체 페이지 공통 문제

1. 페이지 개수 불일치

현재 상태: 페이지 푸터에 "페이지 1/9"로 표시되지만 목차(TOC)에는 8개 페이지만 나열됨. TC 부록이 2페이지(P8-P9)를 차지하므로 동적 합계는 9페이지가 맞음.

해결책: TOC 마지막 항목을 "8-9" 또는 "8+"로 수정하거나 푸터의 전체 페이지 수를 동적으로 계산.

  return '✅ Good';
