Release Readiness 인디케이터 설계

Score신호색상메시지
≥ 80🟢 RELEASE READYRGB(16,163,127)"Project meets quality gates for release"
60~79🟡 CONDITIONALRGB(245,158,11)"Release possible with known risks — review required"
< 60🔴 NOT READYRGB(239,68,68)"Significant quality gaps — release not recommended"

Score 계산은 기존 Overview 탭의 releaseScore 공식을 그대로 사용합니다: (passRate × 0.40) + (critBugResolution × 0.25) + (coverage × 0.20) + (milestoneProgress × 0.15)

차트 그리기 방법 (jsPDF 직접 드로잉)

기술 접근: jsPDF에는 차트 라이브러리가 없으므로, pdf.line()과 pdf.rect()로 직접 그립니다. 30일 데이터를 일별로 집계하여 X축 좌표를 계산하고, Y축은 비율(0~100%)을 mm 단위로 변환합니다.
// Pass Rate 트렌드 차트 그리기 예시 const chartX = margin; const chartY = 40; const chartW = contentW; const chartH = 60; // 60mm 높이 // 일별 데이터 집계 const dailyData = aggregateByDate(rawTestResults, 30); // [{date, passed, total}] dailyData.forEach((day, i) => { const x = chartX + (i / (dailyData.length - 1)) * chartW; const y = chartY + chartH - (day.passRate / 100) * chartH; if (i === 0) pdf.moveTo(x, y); else pdf.lineTo(x, y); }); pdf.setDrawColor(99, 102, 241); pdf.setLineWidth(0.5); pdf.stroke();
