### Fix 1 — TC ID 포맷 변환 함수 (🔴 CEO #1)

**파일**: PDF 생성 코드 (e.g. `src/lib/pdf/generateReport.ts`)

```typescript
// DB에 tc_number 컬럼이 있는 경우 (권장)
function formatTcId(tcNumber: number): string {
  return `TC-${String(tcNumber).padStart(4, '0')}`;
}

// UUID에서 숫자 추출하는 폴백 (tc_number 없을 때)
function formatTcId(rawId: string): string {
  const nums = rawId.replace(/\D/g, '').slice(-4).padStart(4, '0');
  return `TC-${nums}`;
}
```

**적용 위치 (3곳)**:

### Fix 2 — 도형 내 수직 정렬 (html2canvas 대응) (🔴 CEO #2)

### Fix 3 — ▲/▼ 델타 심볼 (🔴 CEO #3)

**3-A. P3 WoW 테이블 Change 컬럼**:
```typescript
// 현재
const changeText = change >= 0 ? `+${change}` : `${change}`;

// 수정
function formatDelta(change: number): { text: string; color: string } {
  return {
    text: change >= 0 ? `▲${Math.abs(change)}` : `▼${Math.abs(change)}`,
    color: change >= 0 ? 'rgb(16,163,127)' : 'rgb(239,68,68)',
  };
}

// HTML 렌더링
const { text, color } = formatDelta(change);
`<td style="color:${color};font-weight:700;">${text}</td>`
```

**3-B. KPI 카드 델타 (P1, P2)**:
```typescript
function renderKpiDelta(delta: number | null, suffix = '%'): string {
  if (delta === null || delta === undefined) return '';
  const symbol = delta >= 0 ? '▲' : '▼';
  const color = delta >= 0 ? 'rgb(16,163,127)' : 'rgb(239,68,68)';
  return `<div style="font-size:8px;color:${color};margin-top:2px;">
    ${symbol}${Math.abs(delta)}${suffix} vs prev period
  </div>`;
}
```

### Fix 5 — TC Appendix Status 필드 교체 (🔴 P8/P9)

**파일**: PDF 템플릿 TC Appendix 렌더링 코드

```typescript
// 현재 (라이프사이클 상태)
const statusText = tc.status; // 'active' | 'draft' | 'deprecated'

// 수정 (실행 결과 상태)
function renderTcStatus(tc: TestCase): string {
  const s = tc.lastExecutionStatus ?? 'untested';
  const map: Record<string, { label: string; color: string; style?: string }> = {
    passed:   { label: 'Passed',   color: 'rgb(16,163,127)', style: 'font-weight:600;' },
    failed:   { label: 'Failed',   color: 'rgb(239,68,68)',  style: 'font-weight:600;' },
    blocked:  { label: 'Blocked',  color: 'rgb(249,115,22)', style: 'font-weight:600;' },
    untested: { label: 'Untested', color: 'rgb(100,116,139)',style: 'font-style:italic;' },
  };
  const { label, color, style = '' } = map[s] ?? map.untested;
  return `<td style="color:${color};${style}">${label}</td>`;
}
```
