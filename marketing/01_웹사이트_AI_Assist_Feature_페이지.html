  • 이전 실행 대비 변화 추이 자동 비교

Flaky Detection AI

불안정한 테스트, AI가 찾아서 Jira까지

같은 테스트가 랜덤하게 성공과 실패를 반복하면 팀의 신뢰를 무너뜨립니다. Flaky Detection AI는 통계적 패턴을 분석하여 불안정한 테스트를 식별하고, 원인 분석 결과를 Jira 이슈로 자동 생성합니다.

  • 불안정 테스트 자동 탐지 및 심각도 분류
  • 근본 원인(환경 의존성, 타이밍, 데이터 등) 자동 분석
  • Jira 이슈 자동 생성 + TC ID 라벨 자동 태깅
  • 주간 Flaky 테스트 트렌드 리포트 제공

Coverage Gap Analysis

  • 누락된 엣지 케이스 및 시나리오 추천
  • 주기적 커버리지 추이 리포트 제공
