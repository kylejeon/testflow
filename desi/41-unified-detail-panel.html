DEV HANDOFF — Unified Detail Panel (TC + Run)
§1 — Problem Statement & Solution

Problem: Steps Crowd Out Tab Content

Symptom7+ steps → Comments/Results tab area shrinks to <100px
Root causeSteps area used max-height: 40vh (fixed) — no awareness of remaining space
User impactCan't read comments or results without collapsing steps

Solution: 3-Layer Space Guarantee

Layer 1Smart max-height: calc()-based, guarantees min 220px for tab body
Layer 2Drag resize handle: user controls steps↔tabs ratio manually
Layer 3Collapse toggle: reduces steps to 0px, maximizes tab area
§2 — Unified Panel Architecture

§3 — Dynamic Steps Height Algorithm
const MIN_TAB_BODY_HEIGHT = 220;  // px — guaranteed minimum
const TABS_BAR_HEIGHT = 36;       // px — tab buttons bar
const TOGGLE_HEIGHT = 34;        // px — steps toggle bar
const RESIZE_HANDLE = 6;         // px — drag handle

function calcMaxStepsHeight(splitEl) {
  const splitH = splitEl.clientHeight;
  const reserved = TOGGLE_HEIGHT + RESIZE_HANDLE
                 + TABS_BAR_HEIGHT + MIN_TAB_BODY_HEIGHT;
  const maxSteps = splitH - reserved;
  return Math.max(80, Math.min(maxSteps, splitH * 0.55));
}
