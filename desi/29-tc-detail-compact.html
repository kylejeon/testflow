Admin user can bulk delete inactive accounts
Verify that admin users with appropriate permissions can select multiple inactive accounts and delete them in a single bulk operation.
Priority
High
Assignee

Alex Park

Created
Mar 25, 2026

Precondition
Admin user is logged in with "Super Admin" role. At least 3 inactive accounts exist in the system (inactive for 90+ days).
Test Steps
1
Navigate to Admin Panel → User Management
2
Apply filter "Status = Inactive" and verify inactive users are listed
3
Select 3 inactive accounts using checkboxes
4
Click "Bulk Actions" → "Delete Selected"
5
Confirm deletion in the confirmation dialog

All 3 selected accounts are removed from the user list
2
Success toast shows "3 accounts deleted successfully"
3
Audit log records the bulk deletion with admin user info
