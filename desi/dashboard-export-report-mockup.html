Dev Note

  • 에러 시 isExporting = false + 실패 Toast

Screen 4 — Toast Alerts (Success / Error)

Export 완료 시 상단 중앙 Toast. 성공=초록, 실패=빨강. Jira toast와 동일 디자인.

SUCCESS PDF / CSV 내보내기 성공 — 3초 후 자동 사라짐
PDF report exported successfully
CSV data exported successfully
ERROR 내보내기 실패 — 5초 후 자동 사라짐 (실패는 더 오래 표시)
Failed to export report. Please try again.
Toast 위치: fixed top-4 left-1/2 -translate-x-1/2 z-50. 성공: bg-[#ECFDF5] + border-[#A7F3D0] + text-[#065F46]. 실패: bg-[#FEF2F2] + border-[#FECACA] + text-[#991B1B].
타이밍: 성공 3초, 실패 5초 자동 사라짐. 닫기 버튼 ri-close-line 우측. Jira auto-create toast와 동일한 디자인 시스템 — slide-down 애니메이션 + fade-out.
Dev Note
  • 기존 useToast() 훅 또는 전역 Toast 컨텍스트 사용
  • PDF: toast.success("PDF report exported successfully")
  • CSV: toast.success("CSV data exported successfully")
  • 실패: toast.error("Failed to export report. Please try again.", { duration: 5000 })
  • 애니메이션: slide-down 진입, fade-out 퇴장 (Jira toast 동일)
