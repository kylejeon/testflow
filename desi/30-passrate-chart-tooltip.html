§4 React Implementation Guide

Recommended Approach

Option A — SVG + HTML tooltip (recommended)
• Render bars as SVG <rect> inside <g class="bar-group">
• Tooltip as HTML div positioned via position: fixed
• Use onMouseEnter / onMouseLeave on <g> to toggle tooltip
• Get bar position via getBoundingClientRect() on the <g> element
• Clamp tooltip X to prevent overflow: Math.max(8, Math.min(x, viewportW - tooltipW - 8))

Option B — Recharts
• Use <Tooltip> component with content={<CustomTooltip />}
• Style CustomTooltip with same dark theme tokens
• Use <Bar> onMouseEnter for highlight effect
