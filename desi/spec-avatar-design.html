1. System Overview

Testably uses Initial Avatars as the default, with optional Photo Upload. Emojis are reserved for project/page icons only (Notion-style), never for user profiles.

Decision: Initial Avatar (default) + Photo Upload (optional) — confirmed by CEO 2026-03-24.

Color Assignment Logic (pseudocode)

StepLogic
1. HashcolorIndex = simpleHash(user.id) % PALETTE.length
2. Initialsinitials = (firstName[0] + lastName[0]).toUpperCase() — if single name, use first 2 chars
3. FallbackIf no name: use email first 2 chars. If nothing: "?" with #94A3B8 (Slate-400)
4. OverrideIf user uploads photo: photo replaces initials; color stored as fallback
