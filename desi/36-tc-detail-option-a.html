§1 — Option A Layout Architecture (Unified with Run Detail)

Panel Structure (top → bottom)

① dp-headerTC-ID + Title + Description + Close — flex-shrink: 0
② dp-meta2-col meta grid (same as Run Detail) — flex-shrink: 0
③ dp-steps-toggleCollapsible toggle bar (same as Run Detail) — flex-shrink: 0
④ dp-steps-area ★max-height: 40vh; overflow-y: auto — flex-shrink: 0
⑤ dp-tabs4 tabs (no Details tab) — flex-shrink: 0
⑥ dp-tab-bodyflex: 1; overflow-y: auto — fills remaining space
⑦ dp-actionsEdit / Delete — flex-shrink: 0 (TC only, Run uses Quick Actions bar)

★ Unified Design — TC vs Run Context

SharedHeader, Meta Grid, Steps Toggle, Steps Area, Tabs, Tab Body — identical CSS
TC onlydp-desc (description), dp-actions (Edit/Delete bar at bottom)
Run onlydp-actions-bar (Quick Actions at top), Step mini buttons, Progress counter
Class namesdp-top, dp-id, dp-title, dp-close — unified across both contexts
Design principle: TC and Run Detail panels share a single base component with context-specific slots. TC context renders description + bottom Edit/Delete bar. Run context renders Quick Actions bar at top + step-level Pass/Fail mini buttons + progress counter in toggle bar. All other elements — header, meta grid, steps, tabs — are pixel-identical.

§4 — React Implementation Guide (Unified Base Component)

★ Unified DetailPanel Component

// Shared base component — context prop drives TC vs Run differences
interface DetailPanelProps {
  context: 'tc' | 'run';
  testCase: TestCase;
  runResult?: RunResult;   // only in Run context
}

const [activeTab, setActiveTab] = useState<'comments' | 'results' | 'issues' | 'history'>('comments');
const [stepsCollapsed, setStepsCollapsed] = useState(false);

Component Structure (JSX)

{""}
<div className="detail-panel">
  <DPHeader description={context === 'tc'} />

  {""}
  {context === 'run' && <QuickActionsBar />}

  <DPMeta />

  {""}
  <StepsToggle
    count={steps.length}
    attachments={attachments.length}
    /* Run context: show progress counter */
    progress={context === 'run' ? { passed, total } : undefined}
    collapsed={stepsCollapsed}
    onToggle={() => setStepsCollapsed(!stepsCollapsed)}
  />

  {""}
  {!stepsCollapsed && (
    <div className="dp-steps-area">
      {precondition && <Precondition />}
      <StepsList steps={steps}
        /* Run only: step-level mini buttons */
        showMiniButtons={context === 'run'} />
      <Attachments />
    </div>
  )}

  <TabBar tabs={['comments','results','issues','history']}
    active={activeTab} onChange={setActiveTab} />

  <div className="dp-tab-body">
    {activeTab === 'comments' && <CommentsTab />}
    {activeTab === 'results' && <ResultsTab />}
    {activeTab === 'issues' && <IssuesTab />}
    {activeTab === 'history' && <HistoryTab />}
  </div>

  {""}
  {context === 'tc' && <DPActions />}
</div>
