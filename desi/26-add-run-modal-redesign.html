Design Annotations — Add Run Modal Redesign

1. 2-Step Wizard 패턴 Step 1 (560px): Run 기본 정보만 집중. Step 2 (840px): TC 선택에 넓은 공간 확보. 단일 모달 안에서 스텝 전환하므로 컨텍스트 유지됨. Step 1에서 "Include all cases" 선택 시 Step 2 스킵 가능 (바로 Create Run).
2. 폴더 탐색: 사이드바 → 가로 탭 좌측 사이드바(w-80) 제거. 가로 스크롤 가능한 folder chip 탭으로 변경하여 TC 테이블 전체 너비 활용. 폴더 아이콘 색상은 기존 14-testcases의 폴더 컬러 매핑 재사용. 전체 카운트가 칩 옆에 표시.
3. 필터: 우측 패널 → 인라인 툴바 Priority/Tags 필터를 드롭다운 버튼으로 축소. 활성 필터 수 표시 (badge count). "Include Draft" 토글을 필터바 우측에 인라인 배치하여 §10 기획 반영. 검색창은 flex-1로 남은 공간 채움.
4. Lifecycle 열 + Draft 행 강조 테이블에 Lifecycle 열 추가 (§1 badge sm 사용). Draft TC 행은 #FFFBEB 배경으로 시각 구분. Deprecated TC는 완전 숨김 — 하단 summary에 "N Deprecated hidden" 표시로 사용자에게 안내.
5. 선택 상태 Summary Bar 하단 summary: 전체 선택 수 (bold) + Active/Draft 내역 breakdown + Deprecated hidden 안내. 기존 "0/31 selected"보다 훨씬 정보 밀도 높음. Draft 경고(§7)는 summary 위에 인라인 표시.
6. Step 1 Form 개선 2-column 레이아웃으로 Milestone+Status, Configuration+Issues 쌍 배치하여 수직 공간 절약. Tags는 chip 입력 방식 (기존 텍스트 → chip). CI/CD 토글은 보라색 하이라이트 섹션으로 시각적 구분. Description은 textarea로 변경.
7. "Include All Cases" 숏컷 Step 1 footer에 "Include all cases" 체크박스 추가 가능 (선택 시 Step 2 스킵하고 바로 Create Run). 이렇게 하면 기존 라디오 버튼 UX를 보존하면서 2-step 플로우도 자연스러움.
8. Animation Spec Modal entrance: translateY(10px) scale(0.98) → normal, 250ms ease-out. Step transition: crossfade (Step 1 out → Step 2 in), 200ms. Draft rows toggle: fadeSlideIn 200ms ease-out. Warning banner: slideDown 200ms. Filter dropdown: §4 dropdown과 동일 패턴.
