색상 배정 알고리즘

// Deterministic color from userId — same color everywhere const AVATAR_COLORS = [ '#6366F1', '#8B5CF6', '#EC4899', '#10B981', '#F59E0B', '#06B6D4', '#EF4444', '#14B8A6', '#F97316', '#3B82F6', ] as const; function getAvatarColor(userId: string): string { let hash = 0; for (let i = 0; i < userId.length; i++) { hash = ((hash << 5) - hash + userId.charCodeAt(i)) | 0; } return AVATAR_COLORS[Math.abs(hash) % AVATAR_COLORS.length]; } function getInitials(fullName: string): string { return fullName .split(' ') .map(w => w[0]) .join('') .toUpperCase() .slice(0, 2); }
핵심 규칙: 색상은 반드시 userId(UUID)로 결정합니다. 이름이나 이니셜 기반이 아닙니다. 같은 사용자는 헤더, 테이블, 댓글, 팀 목록 어디서든 동일한 색상으로 표시됩니다. 이니셜이 같은 다른 사용자(예: "John Kim" JK vs "Jake Kang" JK)도 userId가 다르면 다른 색상이 됩니다.

6 React Component — <Avatar />
Component Interface
interface AvatarProps { user: { id: string; full_name?: string; email?: string; avatar_url?: string | null; }; size: 'xs' | 'sm' | 'md' | 'lg' | 'xl'; status?: 'online' | 'offline' | 'busy' | 'away'; showUpload?: boolean; // XL only className?: string; } function Avatar({ user, size, status, showUpload, className }: AvatarProps) { const initials = getInitials(user.full_name || user.email || '?'); const bgColor = getAvatarColor(user.id); return ( <div className={`av av-${size} ${showUpload ? 'av-upload' : ''} ${className ?? ''}`} style={{ background: user.avatar_url ? 'transparent' : bgColor }} > {user.avatar_url ? <img src={user.avatar_url} alt={initials} /> : initials} {status && <span className={`av-status ${status}`} />} {showUpload && ( <div className="av-overlay"> <i className="ri-camera-line" /> </div> )} </div> ); }

7 Accessibility & Edge Cases
접근성 및 엣지 케이스 처리

Accessibility

aria-label"{full_name}'s avatar" — 모든 아바타에 추가
roleimg — 장식이 아닌 정보 전달 시
색상 대비10색 모두 #fff 텍스트와 WCAG AA 이상 충족
Status dotaria-label="online" 등 상태 정보 포함

Edge Cases

이름 없음"?" 문자 + bg #CBD5E1 (Slate-300)
단일 이름첫 글자만 사용 (예: "Alice" → "A")
3+ 단어 이름첫째 + 마지막 이니셜 (예: "Kim Yong Hyuk" → "KH")
비라틴 문자그대로 표시 (예: "김영혁" → "김영", "太郎" → "太郎")
이미지 로드 실패onError → 이니셜 fallback 자동 전환
Deleted user회색 배경 + 사람 실루엣 SVG
