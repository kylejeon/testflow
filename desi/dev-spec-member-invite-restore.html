🚨 문제 상황: 팀원 초대 기능의 UI 진입 경로가 완전히 끊겨 있습니다.
InviteMemberModal.tsx (509줄)과 ProjectMembersPanel.tsx (346줄)가 존재하고, project-detail/page.tsx에서 import + state 선언까지 되어 있지만, setShowInviteModal(true)를 호출하는 버튼이 없고, ProjectMembersPanel이 JSX에서 렌더링되지 않습니다.
결과: 사용자가 팀원을 초대할 방법이 전혀 없는 상태.

3 Task B: Settings > Members 탭 신설

3-1. 현재 Settings 탭 구조

3-2. Settings Members 탭 요구사항

ID요구사항
B-1탭 추가: activeTab 타입에 'members' 추가. Settings 사이드바에 "Members" 항목 추가. 아이콘: ri-team-line.
B-2탭 콘텐츠: 프로젝트 선택 드롭다운 + ProjectMembersPanel 재사용. 프로젝트를 선택하면 해당 프로젝트의 멤버 목록 표시.
B-3프로젝트 선택 드롭다운: 사용자가 소속된 모든 프로젝트 목록. project_members 테이블에서 현재 유저의 프로젝트 조회.
B-4Invite 버튼: 프로젝트 선택 후 "Invite Member" 버튼 → InviteMemberModal 열기.
B-5URL 지원: /settings?tab=members로 직접 접근 가능. 기존 URL 파라미터 패턴과 동일.

3-3. Settings Members 탭 코드 패턴

파일: src/pages/settings/page.tsx

// ═══ 1. activeTab 타입에 'members' 추가 (line ~89) ═══
const [activeTab, setActiveTab] = useState<
  'general' | 'integrations' | 'notifications' | 'cicd' | 'profile' | 'members'
>('general');

// ═══ 2. URL 파라미터 분기에 'members' 추가 (line ~137) ═══
if (tab === 'members' || tab === 'integrations' || ...) {
  setActiveTab(tab as typeof activeTab);
}

// ═══ 3. 사이드바에 Members 탭 버튼 추가 (General 아래) ═══
<button onClick={() => setActiveTab('members')}
  className={activeTab === 'members' ? '..active..' : '..inactive..'}>
  <i className="ri-team-line" /> Members
</button>
// ═══ 4. Members 탭 콘텐츠 렌더링 ═══
{activeTab === 'members' && (
  <div className="space-y-6">
    <div className="flex items-center justify-between">
      <div>
        <h2 className="text-lg font-bold text-slate-900">Project Members</h2>
        <p className="text-sm text-slate-500 mt-1">
          Manage team members and their roles for each project.
        </p>
      </div>
      <button
        onClick={() => setShowInviteModal(true)}
        disabled={!selectedProjectId}
        className="flex items-center gap-1.5 px-4 py-2 bg-indigo-600
          text-white text-sm font-semibold rounded-lg hover:bg-indigo-700
          disabled:opacity-50 disabled:cursor-not-allowed">
        <i className="ri-user-add-line" /> Invite Member
      </button>
    </div>

    {/* 프로젝트 선택 드롭다운 */}
    <select value={selectedProjectId} onChange={e => setSelectedProjectId(e.target.value)}
      className="w-full max-w-sm h-10 px-3 border rounded-lg text-sm">
      <option value="">Select a project...</option>
      {userProjects.map(p => (
        <option key={p.id} value={p.id}>{p.name}</option>
      ))}
    </select>

    {/* 기존 컴포넌트 재사용 */}
    {selectedProjectId && (
      <ProjectMembersPanel
        projectId={selectedProjectId}
        onInviteClick={() => setShowInviteModal(true)}
        refreshTrigger={memberRefreshTrigger}
      />
    )}
  </div>
)}

6 기존 컴포넌트 Quick Reference

InviteMemberModal Props

PropType설명
isOpenboolean모달 표시 여부
onClose() => void모달 닫기 콜백
projectIdstring대상 프로젝트 ID
onInvited() => void초대 완료 후 콜백 (멤버 목록 갱신용)

내장 기능: 이메일 입력, 이름 입력, 역할 선택 (admin/member/viewer), 구독 티어별 멤버 수 제한 체크, 기존 사용자/신규 사용자 분기, 초대 링크 생성, 알림 + 웹훅 트리거.

ProjectMembersPanel Props

PropType설명
projectIdstring대상 프로젝트 ID
onInviteClick() => void패널 내 "Invite" 버튼 클릭 시 콜백
refreshTriggernumber값 변경 시 멤버 목록 재조회

내장 기능: 멤버 목록 표시 (아바타, 이름, 역할), 역할 변경 드롭다운, 멤버 제거 (확인 다이얼로그), 현재 유저 권한 체크 (admin만 관리 가능).
