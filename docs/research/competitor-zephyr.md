# Competitor: Zephyr (Squad / Scale)

> Slug: `zephyr` · Last researched: 2026-05-13

## 1. Product Snapshot

- **공식 사이트:** https://smartbear.com/test-management/zephyr-scale/ (Scale) / Atlassian Marketplace (Squad)
- **가격 (as of 2026-05):**
  - Free: 10명 이하 Jira 사용자에 한해 무료 (Jira 전체 유저 수 기준)
  - 10명 초과부터 per-Jira-user 과금 (QA 팀원이 아닌 전체 Jira 헤드카운트)
  - 정확한 per-user 단가는 Atlassian Marketplace에서 견적 필요
  - Standard / Advanced 두 에디션 제공
- **무료 플랜:** Jira 10명 이하 한정 무료. 실질적으로 무료 활용 거의 불가
- **트라이얼:** Standard/Advanced 에디션 모두 trial 가능
- **주요 기능:**
  1. Jira native 테스트 케이스 관리
  2. Test cycles & execution tracking
  3. BDD 지원
  4. Traceability (요구사항 → 테스트 → 결과)
  5. 자동화 결과 수집
  6. Advanced: 확장 자동화, 추가 스토리지, API rate 향상
  7. 보고서 및 커버리지 분석
  8. CI/CD 연동 (Jenkins, Bamboo 등)
- **통합:** Jira (필수), Jenkins, Bamboo (일부 지원 종료됨), GitHub Actions (Enterprise만)
- **타겟 고객:** Atlassian 생태계 중심 기업. Jira 없이는 작동 불가.

## 2. User Pain Points (G2 / TrustRadius / Reddit 리뷰)

1. **Jira 전체 유저 과금 — 가장 큰 불만**
   "The biggest con is that you can't do individual licenses for Zephyr Scale making the product very expensive. If you install the plug-in into JIRA then you have to pay for a license for every JIRA user."
   — Source: https://community.smartbear.com/discussions/zephyrscale/pricing/235508

2. **극심한 성능 문제**
   "Reports of 10-20 minute load times and execution screens failing to load for hours."
   — Source: https://aqua-cloud.io/zephyr-review-and-insights-what-users-actually-say-about-this-tms/

3. **고객 지원 최악 수준**
   "One verified reviewer documented three months of back-and-forth with Zephyr support team during a data center to cloud migration, during which the support team repeatedly asked for the same information, failed to answer direct questions."
   — Source: https://aqua-cloud.io/zephyr-review-and-insights-what-users-actually-say-about-this-tms/

4. **마이그레이션 위험**
   "If the Zephyr data transfer fails, teams must delete all data from Jira Cloud and restart entirely."
   — Source: https://www.trustradius.com/products/smartbear-zephyr-scale/reviews

5. **정체된 제품 개발**
   "Verified user feedback in 2025 and 2026 consistently cites stagnant product development as a primary reason teams are exploring Zephyr Scale alternatives."
   — Source: https://www.g2.com/products/zephyr-scale/reviews

## 3. Testably Differentiation (우리 우위 3개)

1. **Jira 독립 플랫폼 — 과금 구조가 근본적으로 다름**
   Zephyr는 Jira 전체 유저에게 과금. Testably는 QA 팀원 수만큼만 flat-rate 과금. 100명 Jira 조직에 5명 QA가 있다면 차이는 극명함. 카피 hook: *"Stop paying Zephyr for 95 people who never test anything."*

2. **Jira 없이도 완전히 작동**
   Testably는 Jira를 선택적 연동으로 제공. Jira를 떠나도 QA 데이터는 보존됨. 카피 hook: *"Your test history shouldn't disappear if you switch issue trackers."*

3. **안정성 + 빠른 온보딩**
   Zephyr는 로딩 10~20분, 마이그레이션 실패 시 전체 초기화. Testably는 5분 내 셋업, CSV 마이그레이션 10분. 카피 hook: *"Set up in 5 minutes. Not 3 months of back-and-forth with support."*
