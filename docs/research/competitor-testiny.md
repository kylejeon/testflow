# Competitor: Testiny

> Slug: `testiny` · Last researched: 2026-05-13

## 1. Product Snapshot

- **공식 사이트:** https://www.testiny.io
- **가격 (as of 2026-05):**
  - Free: 무료, 최대 3명, 5GB 스토리지, 기본 TC·런·플랜·보고 기능, API 키
  - Starter: $18.50/user/month (연간, 1~10명 슬래브), 무제한 이력, 커스텀 필드, 스텝별 결과, CSV/Excel 익스포트, MCP Server 지원, 50GB 스토리지
  - Business: $20.50/user/month (연간), 자동화 결과, 마일스톤, SSO (Google/Microsoft/Auth0/Okta), 프리미엄 지원, 250GB 스토리지
  - Enterprise: $30.00/user/month (최소 5명 패키지), 커스텀 역할·권한 그룹, 감사 로그, 온프레미스(Testiny Server), 500GB 스토리지
  - 연간 결제 시 2개월 무료 (16.7% 할인)
  - 비영리·교육기관 50% 할인, 오픈소스 무료 접근
- **무료 플랜:** 있음 (3명, 기본 기능)
- **트라이얼:** 무료 플랜으로 즉시 시작 가능
- **주요 기능:**
  1. 깔끔한 모던 UI 테스트 케이스 관리
  2. 테스트 플랜 & 런
  3. 자동화 결과 추적 (Business 이상)
  4. 마일스톤 (Business 이상)
  5. Jira, GitHub, GitLab, Azure DevOps 연동
  6. Linear, Asana, Confluence, Trello, monday.com 연동
  7. SSO (Business 이상)
  8. 커스텀 필드 (Starter 이상)
  9. REST API + MCP Server
  10. 온프레미스 옵션 (Testiny Server, Enterprise)
- **통합:** Jira, GitHub, GitLab, Azure DevOps, Redmine, Linear, Asana, Confluence, Trello, monday.com
- **타겟 고객:** 소규모~중규모 팀. 단순하고 빠른 TCM을 원하는 스타트업, 아직 초기 단계인 QA 팀.

## 2. User Pain Points (G2 / Capterra 리뷰)

1. **대용량 TC 처리 시 API rate limit 문제**
   "Users face testing difficulties when exceeding 6900 test cases due to API rate limit messages interrupting workflow."
   — Source: https://www.g2.com/products/testiny/reviews

2. **소화면 해상도에서 정보 잘림**
   "Users face issues with poor resolution, leading to information being cut off on smaller screens during test runs."
   — Source: https://www.g2.com/products/testiny/reviews

3. **부모 폴더에서 직접 런 실행 불가**
   "It would be more convenient to be able to launch a test run directly from the parent folder containing other test case folders."
   — Source: https://www.g2.com/products/testiny/reviews

4. **문서 부족 (셀프서비스 한계)**
   "There's almost no documentation available — though users note the app is pretty self-explaining."
   — Source: https://www.softwareworld.co/software/testiny-reviews/

5. **리포팅 고도화 부족**
   "Users note a desire for more advanced reporting features and integrations."
   — Source: https://www.capterra.com/p/10004572/Testiny/

## 3. Testably Differentiation (우리 우위 3개)

1. **Shared Steps 버전 관리 — Testiny에는 없음**
   Testiny는 Shared Steps 버전 고정 기능 없음. Testably는 버전 고정 + diff + bulk update + run 스냅샷 보존. 리그레션 안정성에서 결정적 차이. 카피 hook: *"Testiny updates every test when you edit a shared step. Testably doesn't."*

2. **AI 테스트 생성 — Testably만 보유**
   Testiny는 AI 기능 없음. Testably는 텍스트 설명·Jira 이슈·탐색 세션 기반 AI 생성 포함. 카피 hook: *"Testiny has great UX. Testably has AI on top of it."*

3. **Flat-rate 팀 플랜 — per-user 모델 대비 유리**
   Testiny Business $20.50/user vs Testably Professional $99/mo (최대 20명 = $4.95/user). 팀 규모 커질수록 차이 극대화. 카피 hook: *"At 10 people, Testiny costs $200+/month. Testably costs $99."*
