# Competitor: TestPad

> Slug: `testpad` · Last researched: 2026-05-13

## 1. Product Snapshot

- **공식 사이트:** https://testpad.com
- **가격 (as of 2026-05):**
  - Free tier: 없음. 30일 full-feature trial (최대 20명 사용 가능, 카드 불필요)
  - Essential: $49/mo (연간) / $59/mo (월간) — 3명
  - Team: $99/mo (연간) / $119/mo (월간) — 10명
  - Team 15: $149/mo (연간) / $179/mo (월간) — 15명
  - Department: $249/mo (연간) / $299/mo (월간) — 25명
  - Enterprise: Custom (맞춤 견적, net30 인보이스)
- **무료 플랜:** 없음 (30일 trial만 제공)
- **트라이얼:** 30일, 전 기능, 최대 20명, 카드 불필요
- **주요 기능:**
  1. 체크리스트 기반 테스트 플랜 (전통적 TC 구조 아님)
  2. 탐색적 테스팅 지원 (SBTM 스타일)
  3. 빠른 TC 생성 및 수정
  4. 이미지 첨부 (Team 이상)
  5. 게스트 테스터 (Team 이상)
  6. API 접근 (Team 이상)
  7. 공유 리포트
  8. 회귀 테스트, UAT 지원
  9. 최소한의 진행률 통계
- **통합:** 제한적. 외부 도구 연동 빈약 (Jira 연동 없음 또는 매우 제한적)
- **타겟 고객:** 소규모 SMB, 비기술적 팀원이 포함된 팀, 가볍고 빠른 테스팅을 원하는 팀

## 2. User Pain Points (G2 / Capterra 리뷰)

1. **통합 기능 매우 부족**
   "In terms of integrations with other apps the support looks limited, if this can be expanded then it's going to very handy for everyone to use Testpad."
   — Source: https://www.g2.com/products/testpad-testpad/reviews

2. **시각화 / 리포팅 제한**
   "The interface only shows tasks in one view and doesn't let users visualize progress in other ways, like charts or graphs."
   — Source: https://www.capterra.com/p/196993/Testpad/

3. **무료 플랜 없음 — trial 종료 후 즉시 유료**
   30일 trial 이후 Essential $49/mo부터 시작. 영구 무료 옵션 없음.
   — Source: https://testpad.com/plans/

4. **체크리스트 방식의 한계 — 복잡한 TC 구조 불가**
   복잡한 단계별 예상 결과, 전제조건, 우선순위 등을 상세히 관리하기 어려운 단순 구조.
   — Source: https://www.g2.com/products/testpad-testpad/reviews

5. **Jira 없는 버그 트래킹**
   내장 이슈 트래커 없음. Jira 연동 기능 미흡.
   — Source: Community feedback on https://testpad.com

## 3. Testably Differentiation (우리 우위 3개)

1. **영구 무료 플랜 + 더 풍부한 기능**
   TestPad는 trial 종료 후 $49/mo 시작. Testably는 영구 무료 플랜(1 프로젝트, 2명, 100 TC) 제공. 그리고 Testably는 Jira 연동, RTM, AI를 훨씬 풍부하게 지원. 카피 hook: *"TestPad charges from day 31. Testably stays free forever."*

2. **Jira two-way 연동 — 전 플랜**
   TestPad는 Jira 연동이 거의 없음. Testably는 Free부터 Jira read, Hobby부터 two-way sync. 카피 hook: *"Testably talks to Jira. TestPad doesn't."*

3. **정형화된 TC + 탐색적 테스팅 모두 지원**
   TestPad는 체크리스트 방식만. Testably는 단계별 TC + 탐색적 테스팅 + AI 생성 모두 지원. 카피 hook: *"Don't choose between structure and flexibility. Testably does both."*
