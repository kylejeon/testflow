# Post-mortem 템플릿

> **작성 기준:** SEV1/SEV2 인시던트 해결 후 48시간 이내 작성.
> 이 문서는 비난 없는(blameless) 회고를 목표로 합니다.
> 파일명 규칙: `post-mortem-YYYY-MM-DD-<짧은-제목>.md` (예: `post-mortem-2026-05-01-db-slowdown.md`)

> 무엇이 왜 발생했는지 기술합니다. "누가 잘못했는가"가 아닌 "어떤 시스템/프로세스가 실패했는가"에 집중합니다.
